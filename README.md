# MIProtocolo

O MIProtocolo é um software desenvolvido para gerar protocolos para envio e recebimento de documentos.

- Cadastro de Nomes de Clientes
- Criação e Gerenciamento de Protocolos
- Editor de Texto para a Descrição dos Protocolos
- Gerador de relatórios em PDF usando o [mPDF](https://github.com/mpdf/mpdf)

## Requerimento

- Sistema Operacional: Linux Debian 12 ou superior, Linux Ubuntu 22.04 ou superior
- Arquitetura: x64
